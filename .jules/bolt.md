## 2024-04-27 - [Optimize .filter().length pattern]
**Learning:** In React components and hooks, using `.filter().length` repeatedly inside a `render` function or when tracking length is O(N) but doing multiple passes `filter` allocations. We can optimize it by reducing passes or avoiding it if we just need counting via `reduce` or `for` loops if performance matters.
**Action:** Replace `array.filter(condition).length` with a `reduce` or `for` loop, especially when there are multiple such operations on the same array in the same render cycle, to improve performance by reducing memory allocation and array traversals.
